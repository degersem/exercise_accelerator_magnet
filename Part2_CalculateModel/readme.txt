Part2_CalculateModel
--------------------

You need FEMM to be installed.

If you did not construct the model yourself, you can
(a) create the model by the script ---model_creation/sis100_geometry.m--- (you need Octave/Matlab);
(b) use the file ---sis100.FEM--- (you do not need Octave/Matlab).

Carry out page 11 of the script.
